# AI Studio Pro

AI Studio Pro is a premium suite of power-user tools for ChatGPT and Gemini that enables instant code block exports, live web previews, and a sophisticated custom prompt library within a polished, professional interface.

## ✨ Features

- **🚀 Instant Code Export**: Save code blocks directly as files with localized extensions (e.g., .py, .js, .rs).
- **🌐 Live Web Previews**: Render HTML, CSS, and SVG code blocks in a draggable, live preview window instantly.
- **📚 Prompt Library**: Access and manage a personal collection of high-efficiency prompts via a global sidebar (`Ctrl+Shift+L`).
- **✍️ Smart Refine**: Re-inject and edit your last prompt into the platform input with a single hotkey (`Ctrl+Shift+R`).
- **📋 Autocopy**: Automatically copy AI-generated code blocks to your clipboard as they complete.
- **🎨 Premium Aesthetic**: A sophisticated, dark-themed "Obsidian & Purple" UI designed for professional focus.

## 🛠️ Installation

### Chrome / Edge / Brave
1. Clone this repository or download the source code.
2. Go to `chrome://extensions/` in your browser.
3. Enable **Developer mode** (top right toggle).
4. Click **Load unpacked** and select the `/chrome` folder from this directory.

### Firefox
1. Go to `about:debugging#/runtime/this-firefox` in Firefox.
2. Click **Load Temporary Add-on...**.
3. Select any file inside the `/firefox` folder from this directory.

## ⌨️ Hotkeys

| Shortcut | Action |
| --- | --- |
| `Ctrl+Shift+L` | Toggle Prompt Library Sidebar |
| `Ctrl+Shift+R` | Refine Last User Prompt |
| `Ctrl+Shift+S` | Save Last Code Block |
