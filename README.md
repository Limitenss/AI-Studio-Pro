# AI Studio Pro

AI Studio Pro is a premium suite of power-user tools for ChatGPT and Gemini. Features include instant code block exports, live web previews, and a custom prompt library within a polished, professional interface.

## Features

- **Instant Code Export**: Save code blocks directly with localized extensions (e.g., .py, .js, .rs).
- **Live Web Previews**: Render HTML, CSS, and SVG code blocks in a draggable preview window.
- **Prompt Library**: Manage a personal collection of high-efficiency prompts via a global sidebar (Ctrl+Shift+L).
- **Smart Refine**: Re-inject and edit your last prompt into the platform input (Ctrl+Shift+R).
- **Autocopy**: Automatically copy AI-generated code blocks to the clipboard on completion.
- **Premium Aesthetic**: A sophisticated Obsidian and Purple UI designed for professional focus.

## Installation

### Chrome / Edge / Brave
1. Go to `chrome://extensions/` in your browser.
2. Enable **Developer mode** (top right).
3. Drag and drop the `.crx` file or click **Load unpacked** and select the `/chrome` folder.

### Firefox
1. Go to `about:debugging#/runtime/this-firefox`.
2. Click **Load Temporary Add-on...**.
3. Select any file inside the `/firefox` folder.

## Hotkeys

| Shortcut | Action |
| --- | --- |
| `Ctrl+Shift+L` | Toggle Prompt Library Sidebar |
| `Ctrl+Shift+R` | Refine Last User Prompt |
| `Ctrl+Shift+S` | Save Last Code Block |
